内置管道操作符

DatePipe 管道操作符

  • {{ today }} 是 {{ today | date: 'yyyy-MM-dd' }}

  • {{ today }} 是 {{ today | date: 'yyyy-MM-dd HH:mm:ss' }}

UpperCasePipe 和 LowerCasePipe 管道操作符

  • {{ username }} 转为大写 {{ username | uppercase }}

  • {{ username }} 转为小写 {{ username | lowercase }}

CurrencyPipe 和 PercentPipe 管道操作符

  • {{ price }} 转为货币形式，默认为美元形式 {{ price | currency }}

    {{ price }} 转为货币形式，人民币形式 {{ price | currency: '￥' }}

  • {{ discount }} 转为百分比 {{ discount | percent }}

JsonPipe 管道操作符

  • team 是一个对象，有 name 和 slogan 属性，实际上这个管道操作符就是 JSON.stringify 操作 {{ team | json }}

自定义管道操作符

  • {{ slogan }} 使用 emoji 操作符之后 {{ slogan | emoji: '😀' : '😁' }}
